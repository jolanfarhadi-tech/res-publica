# Brand Guide

## Purpose
Would state Res Publica's visual and verbal brand identity.

## Background
Only partial fragments exist: Fraunces/Vazirmatn typography and 7 design tokens are referenced in prior architecture material (`brain/KNOWLEDGE/operating-system.md` §4); the Website's own tone reference is the "Stadt-Frankfurt-Anschreiben" (professional, warm, not activist).

**Status: Version 1.0 summary of confirmed fragments only** — a full brand guide (logo usage, color palette, voice examples) is not available and is not invented here.

## Core Principles
Professional, warm, dignified tone; never activist or sensationalist, per confirmed reference material.

## Definitions
Typography: Fraunces (Latin), Vazirmatn (Farsi/RTL). Tone reference: the Stadt-Frankfurt outreach letter.

## Framework
**Version 1.0 proposal, pending approval**, extending the two confirmed fragments (typography, tone reference):
- **Voice:** factual, dignified, warm — never activist, never sensationalist, consistent with the confirmed Stadt-Frankfurt tone reference.
- **Do:** "A Structured Hearing documented this pattern across multiple accounts." **Don't:** "Victims cry out against institutional failure" (sensationalized, not trauma-informed).
- **Do:** "Trust is built through verified, transparent contribution." **Don't:** "Earn your reputation" (violates Zero Gamification and the Trust-never-Reputation rule).
- **Visual proportionality:** imagery and layout should never imply urgency or crisis framing around testimony content, consistent with the no-trigger-imagery rule already established in `../governance/ETHICS_CHARTER.md`.

## Workflow
Any new brand asset (page copy, visual material) is checked against this proposal and `WRITING_STYLE.md` before publication — pending formal approval of this guide itself.
