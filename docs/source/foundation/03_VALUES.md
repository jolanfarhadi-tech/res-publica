# Values

## Purpose
States the organization's 8 standing operating principles precisely, as binding rules rather than aspirational philosophy (see `02_PHILOSOPHY.md` for the "why"; this document is the "what is required").

## Background
Reused directly from the organization's approved Constitution (`brain/CONSTITUTION.md`) — not redefined here.

## Core Principles
1. AI retrieves, translates, connects, and drafts — it never originates an institutional position.
2. Zero gamification, anywhere.
3. Personalization is opt-in; anonymous use is always fully preserved.
4. Success is measured by civic effect, never by attention.
5. The Static Core stays untouched and fast; AI and identity are layered beside it.
6. Offline-first / graceful degradation is a platform-wide property.
7. Trilingual discipline: genuine re-narration, not flat translation.
8. Nonprofit resourcing realism.
