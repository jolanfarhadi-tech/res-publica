# Contributing to Res Publica

Thank you for your interest in contributing to Res Publica.

## Contributions by prior arrangement only

Res Publica requires a Contributor License Agreement (CLA) for
contributions. The canonical CLA text will be published separately.
Until it is available, contributions are accepted only by prior
arrangement with the project maintainers.

Please contact the maintainers through the repository's established
communication channels before preparing or submitting a contribution.
Opening a pull request or otherwise sending code does not by itself mean
that the contribution has been accepted.

The detailed contribution workflow and CLA procedure will be added when
the CLA text has been approved.
